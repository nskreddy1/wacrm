A default admin account has been created for the WACRM application.

## Default Admin Credentials

- **Email:** `admin@wacrm.local`
- **Password:** `AdminPass123!`
- **Role:** Admin

## Important Security Notice

⚠️ **SECURITY WARNING**: This default admin account is for initial setup and testing purposes only.

**You must change the password immediately after first login.**

### Steps to Change Password:

1. Log in with the default credentials
2. Go to **Settings** → **Security** → **Change Password**
3. Enter and confirm your new strong password
4. Save the changes

## What You Can Do With Admin Role

With the admin role, you can:

- ✅ Manage team members (invite, remove, change roles)
- ✅ Configure account settings (WhatsApp integration, message templates, pipelines, tags)
- ✅ Send messages and manage conversations
- ✅ Create and manage contacts, broadcasts, and automations
- ✅ View all account data and analytics
- ✅ Access all features except account deletion/transfer (owner-only)

## Account Roles

The WACRM system uses a role-based access control (RBAC) system:

| Role | Permissions |
|------|------------|
| **Owner** | Full account control, can delete/transfer account, invite/remove members, manage settings |
| **Admin** | Invite/remove members, manage settings, send messages, create contacts/deals |
| **Agent** | Send messages, create contacts, manage conversations, but cannot access settings |
| **Viewer** | Read-only access to all data |

## First Steps After Login

1. **Change the admin password** (see above)
2. Go to **Settings** → **WhatsApp Integration** to connect your WhatsApp Business Account
3. Invite team members if needed via **Settings** → **Team Members**
4. Configure your preferences in **Settings** → **Appearance**
